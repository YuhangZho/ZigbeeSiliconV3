85 // MANUFACTURING CREATORS
86 // The creator codes are here in one list instead of next to their token
87 // definitions so comparison of the codes is easier. The only requirement
88 // on these creator definitions is that they all must be unique. A favorite
89 // method for picking creator codes is to use two ASCII characters in order
90 // to make the codes more memorable. Also, the msb of Stack and Manufacturing
91 // token creator codes is always 1, while the msb of Application token creator
92 // codes is always 0. This distinction allows Application tokens
93 // to freely use the lower 15 bits for creator codes without the risk of
94 // duplicating a Stack or Manufacturing token creator code.
