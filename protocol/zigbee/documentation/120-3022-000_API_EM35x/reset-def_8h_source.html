52 // The reset types of the EM300 series have both a base type and an
53 // extended type. The extended type is a 16-bit value which has the base
54 // type in the upper 8-bits, and the extended classification in the
55 // lower 8-bits
56 // For backwards compatibility with other platforms, only the base type is
57 // returned by the halGetResetInfo() API. For the full extended type, the
58 // halGetExtendedResetInfo() API should be called.
59 
60 // RESET_BASE_DEF macros take the following parameters:
61 // RESET_BASE_DEF(basename, value, string) // description
62 // basename - the name used in the enum definition, expanded as RESET_basename
63 // value - the value of the enum definition
64 // string - the reset string, must be 3 characters
65 // description - a comment describing the cause
66 
67 // RESET_EXT_DEF macros take the following parameters:
68 // RESET_EXT_DEF(basename, extname, extvalue, string) // description
69 // basename - the base name used in the enum definition
70 // extname - the extended name used in the enum definition, expanded as RESET_basename_extname
71 // extvalue - the LSB value of the enum definition, combined with the value of the base value in the MSB
72 // string - the reset string, must be 3 characters
73 // description - a comment describing the cause
