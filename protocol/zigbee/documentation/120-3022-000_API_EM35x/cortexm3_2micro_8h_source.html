62 // The reset types of the EM300 series have both a base type and an
63 // extended type. The extended type is a 16-bit value which has the base
64 // type in the upper 8-bits, and the extended classification in the
65 // lower 8-bits
66 // For backwards compatibility with other platforms, only the base type is
67 // returned by the halGetResetInfo() API. For the full extended type, the
68 // halGetExtendedResetInfo() API should be called.
